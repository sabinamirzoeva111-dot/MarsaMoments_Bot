# Инструкция: как запустить бота-проверку отчётов

У тебя три файла:
- `bot.py` — сам бот
- `knowledge.py` — все правила, цены, проценты (правишь только его, когда что-то меняется)
- `README.md` — этот файл

---

## Шаг 1. Создать бота в Telegram (2 минуты, бесплатно)

1. Открой Telegram, найди **@BotFather**.
2. Отправь `/newbot`, придумай имя и username (должен заканчиваться на `bot`).
3. BotFather пришлёт **токен** — длинная строка вида `123456:ABC-DEF...`. Скопируй её.

---

## Шаг 2. Получить ключ Anthropic

1. Зайди на **console.anthropic.com**, зарегистрируйся.
2. Раздел **API Keys** → создай ключ. Скопируй (показывается один раз).
3. Пополни баланс — для проверки отчётов расход копеечный, $5–10 хватит надолго.

> Это отдельный платный доступ к API. Подписка claude.ai (где мы сейчас) и API —
> разные вещи, ключ нужен именно от console.anthropic.com.

---

## Шаг 3. Вписать ключи

Открой `bot.py`, найди вверху строки:

    TELEGRAM_TOKEN = os.environ.get("TELEGRAM_TOKEN", "СЮДА_ТОКЕН_ОТ_BOTFATHER")
    ANTHROPIC_API_KEY = os.environ.get("ANTHROPIC_API_KEY", "СЮДА_КЛЮЧ_ANTHROPIC")

Замени `СЮДА_ТОКЕН_ОТ_BOTFATHER` и `СЮДА_КЛЮЧ_ANTHROPIC` на свои ключи (в кавычках).

---

## Шаг 4. Установить и запустить

На компьютере/сервере с Python 3.10+:

    pip install python-telegram-bot anthropic
    python bot.py

Если видишь «Бот запущен. Жду отчёты...» — всё работает.

---

## Шаг 5. Добавить бота в группу

1. Добавь бота в нужную группу (например Shifts).
2. **Важно:** напиши @BotFather команду `/setprivacy` → выбери бота → **Disable**.
   Без этого бот видит только команды, а не обычные сообщения с отчётами.
3. Готово. Теперь когда кто-то кидает отчёт — бот реплаем пишет разбор в тот же чат.

---

## Где это держать включённым 24/7

`python bot.py` работает, только пока запущен. Чтобы бот жил постоянно:
- арендуй дешёвый VPS (~$5/мес, напр. Hetzner, DigitalOcean, Timeweb),
- закинь туда эти файлы, выполни шаги 4–5.

Это единственный шаг, где может пригодиться техничный человек — на пару часов.

---

## Что бот проверяет

- Арифметику выручки (Total = Card + Cash, сумма продаж сходится)
- Номер телефона гостя в каждой продаже (правило с 22.05)
- Цены не ниже базовых по точке (ловит несогласованные скидки)
- Серийники принтеров против эталона (расхождение = штраф 100 AED)
- Зарплату фотографа против его процента
- Заполненность всех секций + обе ссылки
- По вторникам напоминает про инвентаризацию и фото принтеров с открытой крышкой

## Если поменялись цены или проценты

Открой `knowledge.py` и поправь нужные строки. Код `bot.py` трогать не надо.
